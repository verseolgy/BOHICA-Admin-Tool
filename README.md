# BOHICA Admin Tool - Ark Dino Spawner

## 🎮 About
BOHICA Admin Tool is a standalone Windows desktop application for managing Ark: Survival Evolved dinosaur spawning and customization. No browser or internet required!

## 🔐 Security
- Password protected with encrypted access
- Runs locally on your PC
- No data sent to external servers

## 📋 Features
- **Capped Dino Spawner** - Generate and customize tamed dinosaurs
- **Creature Database** - 15+ popular dinosaurs pre-loaded
- **Stat Customization** - Health, Stamina, Oxygen, Food, Weight, Melee, Speed
- **Color Presets** - 6 region color customization (10+ colors available)
- **Preset Templates** - Standard, Flex Capped, Triple Capped, Double Capped, Single Capped
- **Command Generator** - Generates spawn commands for server admins
- **One-Click Copy** - Copy generated commands to clipboard

## 🚀 Installation

### Option 1: Download Pre-Built EXE (Easiest)
1. Download `BOHICA-Admin-Tool-1.0.0-Setup.exe` from the Releases
2. Run the installer
3. Launch from your desktop or Start Menu
4. Enter password: `1822`

### Option 2: Build from Source
1. Clone this repository
2. Install Node.js (if not already installed)
3. Run these commands:
   ```bash
   npm install
   npm run build-exe
   ```
4. Find the built EXE in the `dist` folder

## 🔑 Default Password
**Password:** `1822`

## 📖 How to Use

1. **Select a Creature** - Type or click a dinosaur name from the list
2. **Choose Preset** - Select a stat template (Standard, Flex, Triple, etc.)
3. **Customize Stats** - Adjust individual creature stats as needed
4. **Select Colors** - Choose color for each body region (0-5)
5. **Generate Command** - Click "Generate" to create the spawn command
6. **Copy & Paste** - Click "Copy" and paste into your Ark admin console
